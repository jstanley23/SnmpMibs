Catalyst 8540-CSR MIB Support Lists

The below MIBs are supported by the Catalyst 8540-CSR in release 1.0 and later. SNMP version 1 MIBs are in the v1 directory and SNMP version 2 MIBs are in the v2 directory. Note that for most every MIB .my in the v2 directory, there exists a SNMP version 1 conversion of the MIB -V1SMI.my in the v1 directory. The exception is SNMPv2-TC.my. Thanks go to David T. Perkins for his package smicng which was used to produce most of the v1 conversions.

The following support was available in Release 2.0 of the Catalyst 8540-CSR software in addition to Release 1.0.

1. Standard MIBs:

IANAifType-MIB.my

The following support was available in Release 1.0 of the Catalyst 8540-CSR software. The Catalyst 8540-CSR inherits many standard MIBs and Cisco enterprise MIBs as part of the Cisco IOS 11.2 code base.
