Catalyst 2600 MIB Support Lists

The below MIBs are supported by the Catalyst 2600 Token Ring switch.

SNMP version 1 MIBs are in the v1 directory and SNMP version 2 MIBs are in the v2 directory. Note that for most every MIB .my in the v2 directory, there exists a SNMP version 1 conversion of the MIB -V1SMI.my in the v1 directory as well. The exception is SNMPv2-TC.my. The corresponding schema files for SunNet Manager can be found in the schema directory. Oid files for SunNet Manager can be found in the oid directory. Trap files for SunNet Manager can be found in the traps directory.

Thanks go to David T. Perkins for his package smicng which was used to produce most of the v1 conversions.

Supported SNMP version 1 MIBs:

RFC1213-MIB.my
RFC1231-MIB.my
BRIDGE-MIB.my
ETHERLIKE-MIB.my
IF-MIB.my
CAT2600-MIB.my

Supported SNMP version 2 MIBs in V1 format:

DTRCONCENTRATOR-MIB-V1SMI.my
