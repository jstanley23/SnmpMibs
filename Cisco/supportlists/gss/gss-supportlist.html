Global Site Selector (GSS) MIB Support List

The lists below indicates which MIBs and SNMP traps are supported by the Global Site Selector (GSS).

SNMP version 1 MIB files are in the v1 directory and SNMP version 2 MIB files are in the v2 directory.

Note: 1. The maximum access provided by SNMP agent on GSS is read-only for all supported MIB objects.
