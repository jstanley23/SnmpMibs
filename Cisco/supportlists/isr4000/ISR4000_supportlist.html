Cisco ISR 4000 SNMP MIB Support List

The following list indicates the MIBs that are supported by ISR 4000 Routers. SNMP version 1 MIBs are in the v1 directory and SNMP version 2 MIBs are in the v2 directory. Note that for almost every MIB .my file in the v2 directory, there exists a SNMP version 1 conversion of the MIB -V1SMI.my file in the v1 directory. MIBs are categorized into three types: Supported and Verified MIBs, Supported and Unverified MIBs, and Unsupported MIBs.

There are  no new MIBs added for IOS XE 3.8, 3.9, 3.10, 3.11, 3.12 and 3.13 releases. On 3.14 DSL MIBS are added and on 3.15 VTP and STP MIBs are newly added. MIB compliance is being ensured for upcoming hardwares for these releases.
