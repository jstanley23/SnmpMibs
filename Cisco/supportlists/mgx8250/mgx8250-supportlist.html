MGX8250 MIB Support List

The lists below indicate which MIBs and SNMP traps are supported by the MGX8250, MGX8230 and MGX8850 series of switches using
PXM1 as the Controller Card. Depending on the version of the supervisor module software, different MIBs and traps are supported.

SNMP version 1 MIB files are in the v1 directory and SNMP version 2 MIB files are in the v2 directory. Note that for most every MIB .my in the v2 directory, there exists a SNMP version 1 conversion of the MIB-V1SMI.my  in the v1 directory as well.


 Information in this is applicable for Following Products:

     MGX8250
     MGX8230
     MGX8850(PXM1 based controller)
