Catalyst 6000 and Cisco 7600 IOS MIB Support List

The lists below indicate which MIBs and SNMP traps are supported by the Catalyst 6000 and 7600 series of routers (WS-C6006, WS-C6009, WSC-6503, WS-C6503-E, WS-C6504-E, WS-C6506, WS-C6506-E, WS-C6509, WS-C6509-E, WS-C6509-NEB, WS-C6509-NEB-A, WS-C6509-V-E, WS-C6513, ME-C6524GS-8S, ME-C6524GT-8S, OSR-7609, CISCO7603, CISCO7604, CISCO7606, CISCO7609) running Cisco IOS software. Depending on the version of Cisco IOS Software running on Supervisor module, different MIBs and traps are supported.

SNMP version 1 MIB files are in the v1 directory and SNMP version 2 MIB files are in the v2 directory. Note that for most every MIB in the v2 directory, there exists a SNMP version 1 conversion of the MIB in the v1 directory as well. The corresponding schema files for SunNet Manager can be found in the schema directory. Oid files for SunNet Manager can be found in the oid directory. Trap files for SunNet Manager can be found in the traps directory.

Version history is available in wsc6000-supportlist-ios.changes.
