Catalyst 1600 MIB Support Lists

The below MIBs are supported by the Catalyst 1600 Token Ring Switch.

SNMP version 1 MIBs are in the v1 directory and SNMP version 2 MIBs are in the v2 directory. Note that for most every MIB .my in the v2 directory, there exists a SNMP version 1 conversion of the MIB -V1SMI.my in the v1 directory as well. The corresponding schema files for SunNet Manager can be found in the schema directory. Oid files for SunNet Manager can be found in the oid directory. Trap files for SunNet Manager can be found in the traps directory.
