MIBs supported by Cisco Wireless LAN Controller

This page was last updated on July 7, 2010.

MIBs on Cisco Unified Wireless Network Software Release 4.1.155.0 or LATER. The WLC platforms that support this Software Release are 4400, 2000, 2100, 3505, WiSM and 3750 Integrated WLAN Controller. The WLC platform 5500 also supports all the MIBs mentioned in this list, since 6.0.182.0 software release.
