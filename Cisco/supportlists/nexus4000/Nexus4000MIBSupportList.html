Cisco Nexus 4000 MIB Support List


The following lists indicate the MIBs and SNMP notifications supported by Cisco NX-OS software release for the Cisco Nexus 4000 series. Notifications can be enabled through SNMP, or the CLI. See   the Cisco NX-OS Configuration guides at the following website:

http://www.cisco.com/en/US/products/ps9670/tsd_products_support_series_home.html. 

All Cisco NX-OS MIBs are SNMP (SMIv2) MIBs and are located in the v2 directory. Agent capabilities files are listed under their associated MIBs and contain  implementation details on the MIB.  Agent capabilities files may contain multiple AGENT-CAPABILITIES macros. You must refer to the AGENT-CAPABILITIES macro that applies to your Cisco NX-OS software release.
